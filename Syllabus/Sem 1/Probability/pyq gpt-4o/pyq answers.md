   - **Formula**: Skewness = (Q3 + Q1 - 2Q2) / (Q3 - Q1)
   - **Steps**:
     1. Calculate quartiles: Q1, Q2 (median), Q3.
     2. Apply the formula.

2. **Spearman Rank Correlation**
   - **Formula**: rs = 1 - (6 * Σd²) / (n(n² - 1))
   - **Steps**:
     1. Rank data for both variables.
     2. Calculate rank differences (d).
     3. Use the formula.
